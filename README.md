# proxypool
### 可针对特定网站进行扩展，只需在config.py的site中添加指定网站即可
### 稍后即可通过键名获取代理ip
### eg:
```
site = {
  "weibo":"https://weibo.com"  # value是测试代理是否可用的网址
}
```
通过http://127.0.0.1:5000/weibo/1 即可获取一个代理ip
